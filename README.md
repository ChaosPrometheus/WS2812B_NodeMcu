# WS2812B_NodeMcu
мой проект для управление адресной светодиодом по координатом через esp8266


для управление надо выводит команду /on 0 0  0 0 0 0 < яркость от 0 до 255
                                        ^ ^  ^ ^ ^     
                                        x y  ^ ^ синий
                                             ^ зеленый
                                             красный
                                  /off 0 0  0 0 0 0 < яркость от о 255
                                       ^ ^  ^ ^ ^     
                                       x y  ^ ^ синий
                                            ^ зеленый
                                            красный
